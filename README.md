# tarot-journal-v2

\## 🔒 Privacy



Tarot Journal is a client-side web application.



\- All records are stored in LocalStorage.

\- No server or database is used.

\- No analytics or tracking.

\- No account required.

\- You own your data.



If you clear your browser data, your records will be removed.

Please export a backup regularly.

✦ 塔羅覺察日誌 使用說明
這是一個重視隱私的塔羅日誌

本工具不需要註冊、不需要登入、沒有會員系統，也不會將你的占卜紀錄上傳到任何伺服器。

所有資料都儲存在你目前使用的瀏覽器本機。

也就是說：

✅ 我看不到你的資料

✅ 開發者也看不到你的資料

✅ 不會偷偷上傳到雲端

✅ 沒有 AI 偷讀你的占卜內容

你的每一次占卜、覺察筆記、AI 解讀，都只存在你的裝置裡。

但也代表...

只有你自己能保護你的資料。

如果資料遺失，我也沒有任何能力替你找回。

因此建議：

✔ 定期使用【匯出 JSON】備份

✔ 重要紀錄另外保存在雲端（Google Drive、Dropbox 等）

什麼情況下紀錄可能會消失？

這是最多人疑惑的地方。

① 換了不同的瀏覽器

例如：

昨天：

Chrome

今天：

Samsung Internet

或

Firefox

它們是不同資料庫。

紀錄不會互通。

② LINE 裡開啟 ≠ Chrome

很多人不知道：

LINE 點連結

不是一定使用 Chrome。

很多手機會使用：

LINE 內建瀏覽器

或

Android System WebView

這些都會建立自己的本機資料。

例如：

昨天：

LINE 點開工具

↓

今天：

直接打開 Chrome

↓

會發現：

「我的紀錄怎麼不見了？」

其實沒有消失。

只是存在另一個瀏覽器。

建議第一次開啟後，將網址複製到自己平常使用的瀏覽器，再加入書籤或加入主畫面。

③ 換網址

例如：

昨天：

GitHub Pages

https://xxxxx.github.io/

今天：

本機 HTML

file:///...

或

另一個 GitHub 網址

這些都是不同網站。

資料互不相通。

④ 清除瀏覽器資料

如果你：

清除 Cookie
清除網站資料
清除瀏覽資料
使用系統清理 App

都有可能把紀錄一起刪除。

⑤ 無痕模式

如果使用：

Chrome 無痕模式

Safari 私密瀏覽

關閉視窗後，

紀錄通常都會消失。

⑥ 更換手機

因為沒有雲端同步。

換手機時，

請先匯出 JSON。

再於新手機匯入。

⑦ 重設手機

恢復原廠設定

刷機

資料移除

都會一起消失。

⑧ 某些手機系統會自動清除網站資料

少部分 Android ROM

以及 iPhone Safari

如果：

很久沒有使用
手機空間不足
系統清理網站資料

也可能將 IndexedDB 或 Local Storage 清除。

常見問題
Q：為什麼不用雲端？

因為希望你的占卜內容、覺察日誌、內心想法，只屬於你自己。

這個工具刻意設計成：

「離線優先（Offline First）」

沒有帳號。

沒有會員。

沒有伺服器。

也沒有任何人可以閱讀你的紀錄。

Q：如果紀錄不見了，可以幫我恢復嗎？

不能。

因為資料從來沒有離開你的裝置。

也因此，

沒有任何人可以替你找回。

請定期匯出備份。
